# Edge Case Hunter

Enumerate edge cases and boundary conditions.

## What to check
- Empty/null/malformed inputs
- Time zones and clock skew
- Large payloads and limits
- Retries and idempotency

## Prompts
- "List the edge cases for each input field and how they're handled."
- "What happens if this is called twice?"

## Output
- Edge case checklist + tests to add
