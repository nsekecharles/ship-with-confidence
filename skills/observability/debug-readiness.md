# Debug Readiness

Verify the feature can be debugged quickly under pressure.

## What to check
- Metrics for key SLIs (latency, error rate)
- Traces for critical paths
- Runbooks and dashboards
- Feature flags and rollback

## Prompts
- "What signals tell us this is broken?"
- "How do we safely roll back or disable this feature?"

## Output
- Debug readiness checklist
